## 关键发现

### P 品种 `RPP_5D`：前面分钟有差异，最后一秒没差异

- `RPP_5D` 使用 `rolling(window=1200)`（5天）
- 最后一秒 diff=0 → 历史数据尾部正确
- 前面分钟有差异 → **历史边界处有问题**
- 怀疑：`min_cache` 中历史 CSV（到 15:00）和实时 tick 聚合（21:00）之间存在 gap 或重复

### 系统时间在项目中无意义

- `sql_writer_dce.py` 用 `datetime.now()` 作为 tick 日期
- 回放时市场时间 ≠ 系统时间
- 所有调试必须用 **市场时间（valid_index[-1]）**
