## 容易产生判断错误的地方（重点）

1. **`Index.map(lambda)` 返回的是 `Index` 对象，不是 numpy array**
   - 之前用 `pred_res.index.map(lambda x: ...)` 生成 mask，结果 `.sum()` 抛 `AttributeError`
   - 修复：改用 `idx.hour` / `idx.minute` 直接生成 numpy boolean array

2. **Notebook 缓存导致代码更新不生效**
   - 多次修改 `check_factor_realtime.ipynb` 后，Jupyter/VS Code 仍显示旧版本
   - 修复：新建 `check_factor_realtime_v2.ipynb`，避免缓存干扰

3. **`display()` vs `print()` 展示 DataFrame**
   - 完整因子表（228 rows × 211 cols）用 `print()` 会截断且无法滚动
   - 改用 `IPython.display.display()` 才能在 notebook 中分页、滚动查看
