# check_factor.ipynb 使用说明

## 用途
将 `save_files/{symbol}/factors_*.csv` 与研究环境 `all_factor.feather` 按**时间（HH:MM:SS）**对齐，并排显示原始数字，快速定位差异。

## 运行前准备
1. 确保实时测试环境已经生成 `save_files/{symbol}/factors_*.csv`
2. 确保研究环境 `/mnt/Data/writable/liaoyuyang/factor/{symbol}/all_fac/all_factor.feather` 存在

## 使用步骤

### 1. 配置品种和日期（Cell 1）
```python
symbol = "P"                       # 修改为目标品种：P, A, M, Y, C, CS, B, LH
research_date = None               # None=自动匹配；回放20260324时填 "2026-03-24"
```

### 2. 依次运行所有 Cell
从顶部开始，按 `Shift+Enter` 逐个运行即可。

### 3. 重点查看以下 Cell

| Cell | 名称 | 用途 |
|------|------|------|
| 7 | **原始值并排显示** | 选一个时间点，逐因子显示研究值/实时值/绝对差/相对差，一眼看出哪里不一样 |
| 8 | **逐分钟差异矩阵** | 统计每个因子在所有时间点的通过/失败情况，按差异比例排序 |
| 9 | **问题因子详情** | 列出差异比例高、完全不一致的因子，按日级别/跨品种/分钟级分类 |
| 10 | **单因子走势对比** | 选一个因子画图，看两边时间序列是否贴合 |
| 12 | **交互式查看** | 修改 `VIEW_FACTOR` 和 `VIEW_TIME`，看任意因子任意分钟的前后5分钟 |

## 日期映射规则
- `research_date = None`：自动在研究环境找与实时数据**时间范围匹配**的最近交易日
- `research_date = "2026-03-24"`：强制指定研究环境比对日期，适用于回放历史数据场景

## 阈值说明
默认阈值：
- `TOL_ABS = 0.01`：绝对差异 < 0.01 视为一致
- `TOL_REL = 0.05`：相对差异 < 5% 视为一致

日级别因子（如 `day_jump`）在夜盘场景下天然容易对不上，Notebook 会标注分类，请结合原始值人工判断。

## 常见问题

**Q: 研究环境没有该日期数据怎么办？**
A: Notebook 会自动切换到最近的可用日期，并打印提示。

**Q: 为什么某些因子完全对不上？**
A: 先看该因子的分类：
- **日级别因子**（`day_*`）：可能因数据起点不同导致差异
- **跨品种因子**（如 `P_A_cvcorr10_diff`）：涉及其他品种数据对齐，对 ffill 填充敏感
- **分钟级因子**：如果完全不一致，需要重点排查

**Q: 差异比例多少算正常？**
A: 按经验：
- 完全不一致比例 < 10%：正常
- 10% ~ 30%：有一定差异，需排查主要问题因子
- > 30%：差异较大，需检查数据对齐或因子计算逻辑
