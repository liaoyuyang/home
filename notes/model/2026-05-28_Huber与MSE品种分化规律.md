# Huber 与 MSE 损失函数的品种分化规律

## 现象

v1（Huber 损失）上线后，不同品种表现分化严重：

| 品种 | v1 效果 | 原 v0 效果 |
|------|--------|-----------|
| C | **变差** | 原来最好 |
| LH | **变差** | 原来最好 |
| Y | **明显变好** | 原来较差 |
| M | **明显变好** | 原来较差 |
| P | **明显变好** | 原来较差 |
| A | 变好 | 一般 |
| B | 持平/微变 | 一般 |
| CS | 持平 | 一般 |

## 根因分析

通过分析各品种训练集收益率标签的**峰度（kurtosis）**和**极端值比例**，发现核心规律：

**Huber 的好坏取决于：极端值是"信号"还是"噪音"**

### 关键数据

| 品种 | 峰度 | 极端值性质 | MSE 效果 | Huber 效果 |
|------|------|-----------|---------|-----------|
| **C** | **26.3** | 少而精，都是**真信号** | ✅ 强拟合能抓住 | ❌ 线性惩罚放过信号 |
| **LH** | **18.9** | 少而精，都是**真信号** | ✅ 强拟合能抓住 | ❌ 线性惩罚放过信号 |
| CS | 9.8 | 混合 | ➖ | ➖ |
| P | 8.0 | 混合 | ➖ | ➖ |
| A | 5.9 | 噪音偏多 | ➖ | ✅ |
| Y | 5.3 | 噪音偏多 | ❌ 过拟合噪音 | ✅ 鲁棒 |
| M | 5.2 | 噪音偏多 | ❌ 过拟合噪音 | ✅ 鲁棒 |
| B | 4.1 | 最接近正态 | ✅ 理论最优 | ➖ |

### 机制解释

**MSE 的平方放大效应：**
- 对极端值惩罚极重（误差 5% → 损失 25）
- 适合：极端值是**可预测的真信号**（C、LH）
- 不适合：极端值是**噪音**（Y、M）→ 模型被噪音绑架

**Huber 的线性惩罚：**
- 对极端值惩罚较轻（误差 5% → 损失 ~5）
- 适合：极端值是**噪音**（Y、M）→ 不被绑架
- 不适合：极端值是**信号**（C、LH）→ 放过信号

### C 和 LH 的特殊性

- **C**：峰度最高（26.3），标准差最低（0.0007）→ "平时不动，偶尔剧烈跳动"
- **LH**：峰度第二（18.9），无夜盘 → 数据更干净，大行情都是真信号
- 两者的"剧烈跳动"都是**因子可预测的真实行情**，MSE 的强拟合恰好抓住了这些模式

### Y、M 的特殊性

- 峰度低（~5），接近正态
- 但波动频繁，存在大量"伪极端值"
- MSE 过度拟合这些噪音 → 整体 test_corr 差
- Huber 的鲁棒性治好了这个问题

## 结论与 Action

### 损失函数应按品种配置

```python
LOSS_CONFIG = {
    # 极端值是真信号 → MSE 强拟合
    'C':  {'objective': 'regression', 'metric': 'rmse'},
    'LH': {'objective': 'regression', 'metric': 'rmse'},
    
    # 噪音多 → Huber 鲁棒
    'Y':  {'objective': 'huber', 'metric': 'l1', 'alpha': 0.9},
    'M':  {'objective': 'huber', 'metric': 'l1', 'alpha': 0.9},
    'A':  {'objective': 'huber', 'metric': 'l1', 'alpha': 0.9},
    
    # 混合/正态 → 默认 Huber 保守处理
    'P':  {'objective': 'huber', 'metric': 'l1', 'alpha': 0.9},
    'CS': {'objective': 'huber', 'metric': 'l1', 'alpha': 0.9},
    'B':  {'objective': 'huber', 'metric': 'l1', 'alpha': 0.9},
}
```

### 验证方式

下次换训练截止日期时，先用 `analyze_label_distribution.py` 跑一遍峰度：
- 峰度 > 15：优先考虑 MSE
- 峰度 5~15：混合，按回测效果选
- 峰度 < 5：优先考虑 Huber
